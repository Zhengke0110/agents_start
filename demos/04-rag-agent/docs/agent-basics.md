# Agent 基础概念

## 什么是 Agent？

Agent（智能体）是一个能够自主感知环境、做出决策、并采取行动的程序。

## Agent 的核心三要素

1. **感知（Perception）**：Agent 通过输入（用户消息、API 返回、传感器数据）感知环境状态。
2. **推理（Reasoning）**：Agent 使用大语言模型分析当前信息，决定下一步做什么。
3. **行动（Action）**：Agent 执行具体操作——调用工具、生成回复、修改文件等。

## Agent 与普通 LLM 的区别

- 普通 LLM 只能根据训练数据生成文本，不知道当前时间，不会使用外部工具。
- Agent 是 LLM + 工具 + 决策循环，可以获取实时信息、执行操作。

## ReAct 模式

ReAct = Reasoning + Acting。Agent 交替进行推理和行动：
- 推理：分析现状，决定调用哪个工具
- 行动：执行工具，获取结果
- 观察：分析工具结果，决定下一步

这个过程循环进行，直到任务完成。
