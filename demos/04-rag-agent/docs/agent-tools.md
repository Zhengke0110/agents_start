# Agent 工具系统

## 什么是工具？

工具是 Agent 可以调用的外部函数。每个工具提供一种特定能力。

## 常见工具类型

1. **计算工具**：执行数学运算、单位转换。
2. **信息查询工具**：查询天气、股票、新闻等实时信息。
3. **文件系统工具**：读写本地文件、列出目录。
4. **网络工具**：搜索网页、访问 API。
5. **数据库工具**：查询和操作数据库。

## 工具 Schema

工具 Schema 是用 JSON 格式描述工具的"说明书"。内容包括：

- name：工具名称
- description：工具功能描述
- parameters：需要的参数及其类型

## 工具调用的安全原则

1. **最小权限**：只给 Agent 完成任务所需的最小权限。
2. **路径沙箱**：文件操作限制在指定目录内。
3. **输入验证**：不信任模型传入的参数，必须做安全检查。
4. **用户确认**：危险操作（删除文件、发送邮件）前需要人类审批。
